# 🚀 Code Vault: Personal Snippet Manager & AI Assistant

Code Vault is a SaaS-style web application built to help developers store, manage, and share their code snippets efficiently. It integrates a powerful AI assistant to generate code on the fly and features a RESTful API, making it a step toward a fully Cloud-Native architecture.

## ✨ Key Features

* **🤖 AI Coding Assistant:** Integrated with Google's Gemini 2.5 Flash API to answer coding questions and automatically save AI-generated snippets to the vault.
* **🌍 Public Sharing (Public Links):** Generate secure, unique URLs to share specific snippets with anyone, even without requiring them to log in.
* **📊 Analytics Dashboard:** A visual dashboard using `Chart.js` to track snippet statistics and language distribution.
* **⚡ Live Smart Search:** A real-time autocomplete search feature powered by a custom REST API, providing instant results without page reloads.
* **🏷️ Dynamic Language Support:** Users can choose from predefined languages (C#, Java, Kotlin, SQL, etc.) or seamlessly add custom languages on the fly.
* **🌙 Developer-First UI:** A sleek, responsive Dark Theme built with Bootstrap (Cyborg) and Highlight.js for beautiful syntax highlighting.

## 🛠️ Tech Stack

* **Backend:** C#, ASP.NET Core MVC, Entity Framework Core
* **Database:** SQL Server
* **Frontend:** HTML5, CSS3, JavaScript, Bootstrap 5
* **APIs & Integrations:** Google Generative AI (Gemini 2.5 Flash), RESTful APIs (Controllers)
* **Libraries:** Chart.js (Data Visualization), SweetAlert2 (Notifications), Highlight.js (Code Formatting)
